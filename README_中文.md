# FModel 中文版使用说明

FModel 中文版是一款面向中文用户的 Unreal Engine 游戏资源浏览工具。它基于原版 FModel 制作，补充了简体中文界面，方便用户浏览、预览、搜索和导出 UE4 / UE5 游戏资源包。

如果这个项目帮到了你，欢迎给本仓库点一个 Star。

## 主要功能

- 浏览 UE4 / UE5 游戏资源包
- 搜索资源路径、文件名和资源类型
- 预览贴图、模型、动画、音频等资源
- 导出资源文件
- 管理 AES 密钥和映射文件
- 使用 3D 查看器查看模型资源
- 使用音频播放器预览声音资源

## 适合谁使用

- 想查看 UE4 / UE5 游戏资源结构的玩家
- 需要导出贴图、模型、音频或文本的汉化/Mod 制作者
- 想研究游戏资源组织方式的学习者
- 希望使用中文界面的 FModel 用户

## 下载与运行

如果你下载的是已经编译好的版本：

1. 解压压缩包。
2. 运行 `FModel.exe`。
3. 首次启动后，根据提示选择游戏资源目录。

如果 Windows 提示未知发布者，请确认文件来源可信后再运行。

## 基本使用流程

1. 打开 FModel。
2. 在目录选择器中添加游戏资源目录，通常是游戏安装目录下的 `Content\Paks`。
3. 选择合适的 UE 版本或游戏类型。
4. 点击加载。
5. 在左侧目录树或资源列表中浏览文件。
6. 双击资源进行预览，或右键选择导出。

## AES 密钥

部分游戏资源包经过加密，需要 AES 密钥才能打开。

如果资源包显示为加密或无法读取：

1. 打开 AES 管理器。
2. 填入对应游戏的 AES 密钥。
3. 保存后重新加载资源包。

密钥通常以 `0x` 开头，后面跟 64 个十六进制字符。

## 映射文件

部分 UE4 / UE5 游戏需要映射文件才能正确解析蓝图、结构体或复杂资源。

常见映射文件扩展名为：

```text
.usmap
```

在设置中选择对应游戏版本的映射文件后，再重新加载资源包即可。

## 资源导出

选中资源后，可以通过右键菜单导出。

常见导出内容包括：

- 贴图
- 音频
- 模型
- 动画
- 数据表
- 文本资源

导出的文件会保存到 FModel 的输出目录中。请注意，导出的游戏资源通常仍受原游戏版权保护，不要随意公开传播。

## 3D 查看器

3D 查看器可用于查看模型、骨骼和部分动画资源。

常用操作包括：

- 旋转视角
- 缩放视角
- 移动视角
- 切换材质、贴图或光照相关选项

不同游戏的资源格式差异较大，如果某些模型无法正常显示，通常需要映射文件、正确的 UE 版本，或等待 FModel / CUE4Parse 后续支持。

## 常见问题

### 加载后资源为空

请检查：

- 游戏目录是否选到了正确的 `Paks` 目录
- UE 版本是否选择正确
- 资源包是否加密
- 是否需要 AES 密钥

### 资源显示乱码或解析失败

请检查：

- 是否需要映射文件
- 映射文件是否匹配当前游戏版本
- 游戏是否使用了特殊加密或自定义资源格式

### 3D 模型无法打开

可能原因：

- 当前资源类型暂不支持
- 缺少映射文件
- 游戏使用了自定义模型格式
- 材质或贴图依赖没有一起加载

### 音频无法播放

可能原因：

- 音频格式暂不支持
- 资源包未完全加载
- 需要额外的音频解码组件

## 自行编译

如果你下载的是源码，可以自行编译。

需要准备：

- Windows
- Git
- .NET SDK

克隆源码：

```powershell
git clone --recurse-submodules <仓库地址>
cd <仓库目录>
```

如果已经克隆但没有拉取子模块：

```powershell
git submodule update --init --recursive
```

编译：

```powershell
dotnet publish .\FModel\FModel.csproj -c Release -r win-x64 --self-contained false
```

编译完成后，程序通常位于：

```text
FModel\bin\Release\net10.0-windows\win-x64\publish
```

实际路径可能会随项目版本和 .NET SDK 版本略有不同。
